These represents what kind of press the user has done with the button: * Flag = 0: no press * Flag = 1: short press * Flag = 2: long press * Flag = 3: double click When used in any process, the flags output by the button are meant to be kept track of, and in every cycle, the current flag should be compared to the old flag; the difference between them determines the button delta, which, when not equal to zero, indicates that something (depending on the integrated system) should happen.
